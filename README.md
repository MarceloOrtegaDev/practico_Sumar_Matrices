# Para utilizar este proyecto debes:
- Clonar el repositorio: "https://github.com/MarceloOrtegaDev/practico_Sumar_Matrices.git"
- Utilizar uno de los backends

  ## Python
  ```
  Cd backendPy
  python -m venv venv
  pip install
  python app.py
  ```

  ## Javascript
  ```
  cd backend
  npm install
  npm run dev
  ```

## Frontend
```
npm install
npm run dev

```
